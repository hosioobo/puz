macOS menu bar app · v0.2.1 release

Movement breaks that interrupt you back.

puz is a tiny macOS app for fullscreen routine prompts and countdowns. It is built for breaks you actually want to do, not reminders you instantly dismiss.

Download macOS zip View source 한국어 설명

Fullscreen by design

A routine prompt appears over your work. Once started, the countdown covers every connected display until you press Resume.

Small schedule model

Use multiple routines with weekdays, time windows, run counts, gaps, and predictable or stable-random distribution.

Local-first

Settings and completion records stay on your Mac. No account, server, or analytics pipeline is part of this checkpoint.

What v0.2.1 includes

  • Menu bar app with a compact <//> identity.
  • Multi-routine scheduling with virtual slots and local completion/skip records.
  • Start prompt with short, long, and random snooze options.
  • Fullscreen countdown and Resume-required completion.
  • English/Korean app copy selected from the system-preferred language.
  • No system notification permission request in the current app path.

Download

Download puz-macos.zip from GitHub Releases, unzip it, then open puz.app. The app is unsigned/not notarized for now, so macOS may ask for manual first-open approval.

Build from source

git clone https://github.com/hosioobo/puz.git
cd puz
swift run PauseCoreTestRunner
swift build --product PauseApp

쉽게 넘기지 못하는 운동/휴식 알림.

puz는 전체화면 루틴 프롬프트와 카운트다운을 띄우는 작은 macOS 앱입니다. 그냥 사라지는 알림이 아니라, 잠깐 멈추고 움직이게 만드는 인터럽트를 목표로 합니다.

한국어 README GitHub에서 보기 English

전체화면 중심

시작 프롬프트가 작업 위에 뜨고, 시작하면 모든 모니터를 덮는 카운트다운이 표시됩니다.

단순한 일정

여러 루틴이 요일, 시간 구간, 하루 횟수, 최소 간격, 고르게/안정적 랜덤 배치를 기준으로 virtual slot을 만듭니다.

언어 기반

시스템 선호 언어에 따라 영어/한국어 앱 문구를 선택하고, 미지원 언어는 영어로 fallback합니다.

로컬 우선

설정과 완료 기록은 Mac에 저장됩니다. 계정, 서버, 분석 파이프라인은 없습니다.
